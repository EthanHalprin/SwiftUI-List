# SwiftUI-List
Example for a List in SwiftUI. Content is loaded dynamically from a file server. Images are fetched and cached.

Features:

* MVVM

* Combine Networking

* AsyncImage

* Image cache

* Image Placholders

* Timeout

* Refresh Button

* ProgressView

[Pre-Requisities: iOS 15 Xcode 13]
